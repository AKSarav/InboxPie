# Code of Conduct

InboxPie aims to be a welcoming project for users and contributors.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Keep discussions focused on improving the project.
- Be considerate when discussing privacy and mailbox data.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Publishing private user data or mailbox details
- Repeated off-topic or disruptive behavior
- Threats or abusive language

## Reporting

If you experience or observe unacceptable behavior, report it to the project maintainer. If no private contact is available yet, open a GitHub issue asking for a private reporting channel without including sensitive details.

## Enforcement

The maintainer may remove comments, close issues, reject contributions, or block users who violate this code of conduct.
